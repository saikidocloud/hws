AWS Certified Cloud Practitioner (CLF-C02)

概要

  • AWS クラウドの価値と責任共有モデルの理解
  • セキュリティ・ベストプラクティス・Well-Architected フレームワーク
  • 主要サービス概要・ユースケース・料金体系

対象者

クラウド未経験または経験6ヶ月以下。AWS利用職種に携わる方。

試験形式

  • 択一・複数選択問題
  • 出題数：スコア対象50問＋非採点15問
  • 合格スコア：700 / 1000

出題分野と配分

  • クラウドのコンセプト：24%
  • セキュリティとコンプライアンス：30%
  • クラウドテクノロジーとサービス：34%
  • 請求、料金、サポート：12%

主なタスクステートメント（抜粋）

クラウドのコンセプト

  • AWSの利点とCAF
  • Well-Architected フレームワークの柱
  • クラウドエコノミクスの理解

セキュリティとコンプライアンス

  • 責任共有モデルとIAM
  • コンプライアンス要件・CloudTrail, Security Hub等

クラウドテクノロジーとサービス

  • グローバルインフラ（リージョン、AZ、エッジ）
  • 主要サービス：EC2、S3、Lambda、RDS、VPC等
  • AI/ML、分析、アプリ統合

請求、料金、サポート

  • 料金モデル（オンデマンド、RI、SP）
  • コスト管理ツール：Budgets, Cost Explorer
  • サポートプラン・AWS公式リソース

主要サービスカテゴリ例

  • コンピューティング: EC2, Lambda, Elastic Beanstalk
  • ストレージ: S3, EBS, Glacier, Storage Gateway
  • ネットワーキング: VPC, Route 53, Direct Connect
  • セキュリティ: IAM, KMS, WAF, Shield
  • データベース: RDS, DynamoDB, ElastiCache

試験対象外の業務

  • コーディング
  • アーキテクチャ設計・実装
  • トラブルシューティング
  • パフォーマンステスト
